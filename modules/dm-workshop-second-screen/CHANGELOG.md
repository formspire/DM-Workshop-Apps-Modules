# Changelog

## v0.3.0-alpha.3

- Adds live view-box sync from the GM Player View Box to the clean second display through Foundry sockets.
- Opens the Second Screen action as a live clean Foundry display window instead of a static stream relay.
- Keeps the Player View Box available as a draggable/resizable control frame for what players see.
- Enables module socket support for cross-window display control.

## v0.3.0-alpha.2

- Opens the blank second-screen window immediately from the launcher button click to avoid browser popup timing blocks.
- Adds a manual open prompt with a direct second-screen link if the browser still blocks automatic window creation.

## v0.3.0-alpha.1

Initial public alpha release.

- Player display window
- Second screen support
- Camera follow
- Monitor selection groundwork
- Documentation
- Repository setup
